Step 1 of 6

Open Your Google Sheet

Open the sheet where you want RSVPs to land. Make sure Row 1 has these exact column headers:

household | guests | guest_count | ceremony_count | address | mailing_block | phone | save_date_status | rsvp_status | attending | adults_attending | kids_attending | food_choices | dietary_notes | drinkers | tags | notes

Headers must be lowercase and match exactly. Extra columns are fine — the script only writes to columns it recognizes.

Step 2 of 6

Open Apps Script

In your Google Sheet:

  1. Click Extensions in the top menu bar
  2. Click Apps Script
  3. A new tab opens with a code editor
  4. You'll see a file called Code.gs with an empty myFunction()
Step 3 of 6

Paste the Script

  1. Select all the existing code (Cmd+A) and delete it
  2. Copy the code below and paste it in
  3. Hit Cmd+S to save
function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var headers = sheet.getRange(1, 1, 1, sheet.getLastColumn()).getValues()[0]; var data = e.parameter; var row = headers.map(function(header) { var key = header.toString().trim().toLowerCase(); return data[key] !== undefined ? data[key] : ''; }); sheet.appendRow(row); return ContentService .createTextOutput(JSON.stringify({ result: 'ok' })) .setMimeType(ContentService.MimeType.JSON); } function doGet(e) { return ContentService .createTextOutput(JSON.stringify({ status: 'RSVP webhook is live' })) .setMimeType(ContentService.MimeType.JSON); }
Step 4 of 6

Deploy as Web App

  1. Click the blue Deploy button (top right)
  2. Select New deployment
  3. Click the gear icon ⚙️ next to "Select type"
  4. Choose Web app
  5. Set Execute as: → Me
  6. Set Who has access: → Anyone
  7. Click Deploy
  8. It will ask you to authorize — click through all the permissions (you may need to click "Advanced" → "Go to Untitled project")

The "Anyone" access is safe — it only allows adding rows to your sheet, nothing else.
